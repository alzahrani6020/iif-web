# محرك البحث SearXNG — **مشروع مستقل**

> **لا يُدمج حالياً** مع صفحات الصندوق أو المنصة الحكومية. نبني المحرك ونختبره هنا حتى ننتهي، ثم نربطه لاحقاً عند الحاجة.

## التشغيل

من **هذا المجلد** (`engines/searxng/`):

```bash
docker compose up -d
```

- **واجهة المحرك:** [http://127.0.0.1:8080](http://127.0.0.1:8080)
- يتطلب **Docker Desktop** شغّالاً.

## الإيقاف

```bash
docker compose down
```
