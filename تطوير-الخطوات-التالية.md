## الخطوة التالية المقترحة (جلسة عمل واحدة)

1. تشغيل البيئة: Docker Desktop → `engines/searxng` → `docker compose up -d` (يشغّل Tor + SearXNG) → من الجذر `npm start`.  
2. اختيار صف واحد من الجدول أدناه حسب الأولوية (غالباً **1** أو **2**).  
3. بعد التعديل: `npm run verify` ثم (إن وُجد Git) commit واضح.
