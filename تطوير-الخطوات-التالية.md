## التالي (اختر حسب الحاجة)

| الأولوية | المهمة | ملاحظة |
|----------|--------|--------|
| **1** | ضبط **SearXNG** (محركات، لغة، خصوصية) | حسب [خطة التطوير](engines/searxng/خطة-التطوير.md) المرحلة 2 |
| **2** | **بيانات الجهات** | تعديل `government-data.json` ثم `npm run build` |
| **3** | **واجهة الصندوق** `iif-fund-demo/` | محتوى وتجربة مستخدم حسب المنتج |
| **4** | **نشر إنتاج** | Netlify مربوط بـ `iif-web`؛ بحث الويب على العام يحتاج خادماً لاحقاً |
| **5** | **Netlify + بحث ويب** | دالة Netlify أو خادم خارجي لـ SearXNG — لاحقاً |

## أوامر يومية

```bash
npm start                    # خادم
npm run check-urls           # بعد التشغيل — تحقق من الروابط
npm run build                # بعد تعديل JSON الجهات
```
