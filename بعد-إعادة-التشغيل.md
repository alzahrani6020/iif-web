# بعد إعادة تشغيل الجهاز — تكملة العمل

## 1) Docker Desktop
- شغّل **Docker Desktop** من قائمة ابدأ وانتظر حتى يظهر **Running** (قد يستغرق دقيقة أو أكثر).

## 2) SearXNG (محرك البحث المحلي)
في **PowerShell** أو **CMD**:

```text
cd C:\Users\vip\iif-fund-demo\engines\searxng
docker compose up -d
```

أو من مستكشف الملفات: **`engines\searxng\start-searxng.bat`** (نقر مزدوج) — يشغّل **Tor** + **SearXNG**.

- الواجهة: **http://127.0.0.1:8080**

## 3) خادم التطوير (المنصة + بروكسي `/api/searx`)
من جذر المشروع:

```text
cd C:\Users\vip\iif-fund-demo
npm start
```

- الواجهة: **http://127.0.0.1:3333/**  
- البروكسي يوجّه إلى SearXNG على المنفذ **8080** (تأكد أن الحاوية تعمل كما في الخطوة 2).

## 4) تحقق سريع (اختياري)

```text
docker --version
docker compose ps
```
