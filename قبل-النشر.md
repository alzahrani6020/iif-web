# قبل النشر — قائمة جاهزية المحتوى والتقنية

> **مبدأ:** انشر عندما يكون **المحتوى** مرضياً؛ النشر التقني (Netlify + GitHub) جاهز عند الحاجة.

استخدم هذه القائمة كـ **تعريف «جاهز للنشر»** لنسخة عامة. طبّقها محلياً ثم ادفع إلى `main`.

**ملخص ما أُعدّ في المستودع (هيكل + وثائق):** [PREPUBLISH-COMPLETE.md](./PREPUBLISH-COMPLETE.md)

---

## أ) المحتوى والمنتج

- [ ] **بيانات الجهات:** `government-data.json` محدّثة؛ ثم `npm run build` (أو `npm run verify`).
- [ ] **المنصة الحكومية:** مراجعة نصوص وواجهة `SIMPLE-GOVERNMENT-PLATFORM.html` (عناوين، فلاتر، رسائل للمستخدم).
- [ ] **واجهة الصندوق:** الصفحة المعتمدة (مثلاً `financial-consulting/iif-fund-demo/index.html`) — روابط تعمل، لا صفحات «قريباً» غير مقصودة.
- [ ] **لوحة التحكم ووضع الإدارة:** بعد `npm start` نفّذ `npm run smoke:html`؛ راجع القائمة التفصيلية [financial-consulting/iif-fund-demo/QA-PRE-RELEASE.md](./financial-consulting/iif-fund-demo/QA-PRE-RELEASE.md).
- [ ] **روابط خارجية:** عيّنتَ `website` لكل جهة؛ تتوقّع أن بعض المواقع الحكومية قد تحجب من شبكات معيّنة (لا خطأ من المشروع).
- [x] **صفحات ثقة (مُضافة في المشروع):** مجلد **`legal/`** — فهرس، [خصوصية](./legal/privacy.html)، [إخلاء مسؤولية](./legal/disclaimer.html)، [تواصل](./legal/contact.html). اختصارات: `/privacy`، `/disclaimer`، `/contact`، `/legal` (انظر `netlify.toml`). **راجع النصوص** وحدّث بيانات التواصل في `legal/contact.html` إن لزم.

---

## ب) التقنية قبل `git push`

- [ ] **`npm run verify`** ينجح من جذر المشروع (بناء + فحص السكربتات).
- [ ] **`npm run check-urls`** بعد تشغيل `npm start` (يتحقق من `/` وواجهة الصندوق والمنصة و`fund-site` — راجع `scripts/check-urls.js`).
- [ ] لا توجد **أسرار** أو مفاتيح حقيقية في الملفات المرفوعة — راجع [SECURITY.md](./SECURITY.md).
- [ ] `engines/searxng/config/settings.yml`: **`secret_key`** ليس للإنتاج العلني كما هو؛ غيّره قبل أي مثيل SearXNG معرّض للإنترنت — إرشادات: [engines/searxng/SECRET-PRODUCTION.md](./engines/searxng/SECRET-PRODUCTION.md).
- [ ] إن نشرت **SearXNG + Tor** للعام: راجع المخاطر القانونية/التشغيلية؛ الإعداد المحلي (Docker) للتطوير لا يعادل سياسة إنتاج بدون مراجعة.

## ج) توقّعات الإنتاج (Netlify)

- [ ] فهمت أن **بحث الويب عبر SearXNG** على النطاق العام **لا يعمل افتراضياً** (لا بروكسي على Netlify)؛ **بحث الجهات محلي** يعمل. التفاصيل: [README-IIF-INTEGRATION.md](./financial-consulting/government-search/README-IIF-INTEGRATION.md).
- [ ] **النشر الفعلي:** ربط المستودع [alzahrani6020/iif-web](https://github.com/alzahrani6020/iif-web) بـ Netlify، فرع **`main`**. خطوات اللوحة: [NETLIFY-CHECKLIST.md](./NETLIFY-CHECKLIST.md).
- [ ] بعد النشر: جرّب `/` و **`/fund`** و **`/gov`** و **`/legal`** و **`/privacy`** (إن وُجدت في `netlify.toml`).
