# مسار المستخدم المقترح (تدريب داخلي)

1. **زائر جديد** — الصفحة الرئيسية → يقرأ المهمة والقطاعات → **لماذا الصندوق** / **الثقة**.
2. **جهة حكومية أو مستثمر** — **اتصل بنا** أو **طلب تمويل** / **دراسة جدوى** حسب القسم.
3. **تسجيل** — إنشاء حساب → تسجيل الدخول → **ملفي الشخصي** (الهيدر) لتحديث البيانات والصورة.
4. **عضوية مدفوعة / صلاحيات** — بعد الموافقة: **لوحة التحكم** من الرابط المعروض لمن يملك الصلاحية.
5. **بحث خارجي** — أزرار **محرك البحث** (SearXNG) و**بحث Google** تفتح مواقع طرف ثالث؛ النتائج ليست من خادم الصندوق.

للمراجعة بعد كل نشر: افتح الموقع على **هاتف حقيقي** + سطح مكتب، ولغتين (عربي/إنجليزي) على الأقل.
