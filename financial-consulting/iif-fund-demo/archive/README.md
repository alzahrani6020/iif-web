# أرشيف — نسخ قديمة من الواجهة

هذا المجلد يحتوي نسخاً احتياطية ومتغيرات سابقة من `index.html` (مثل `index-premium.html`, `index-modern.html`, …).

- **لا تُستخدم في النشر الحالي.**
- مسارات CSS/JS فيها قد تشير إلى ملفات لم تعد بجانبها في هذا المجلد.
- فحص الصيانة (`maintenance-full-audit.mjs`) **يتجاهل** هذا المجلد عند تدقيق الروابط حتى لا تُخلط بالمسار الإنتاجي.

الواجهة المعتمدة: **`../index.html`** في المجلد الأعلى.
