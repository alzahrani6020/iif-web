Web search

On the main page, use the header search box for in-site lookup. For a self-hosted meta-search stack, run the repo dev server and open SearXNG per SITE-OPS.md (local default often http://127.0.0.1:18080/).

Government entity lookup lives under the government-search flows linked from the main site.
