# استخدام الطبقة المجانية (Netlify + GitHub) بدون إهدار

> الهدف: **أقل عمليات بناء ممكنة** مع بقاء الموقع محدّثاً عند التغييرات المهمة.

---

## 1) Netlify — ماذا يستهلك الحصة؟

| العنصر | ملاحظة |
|--------|--------|
| **دقائق البناء (Build minutes)** | كل مرة يعمل فيها «Deploy» (إنتاج، معاينة PR، فروع…) |
| **عرض النطاق الترددي (Bandwidth)** | زيارات الموقع — الصفحات الثابتة الخفيفة عادة **قليلة التكلفة** |
| **طلبات الدوال (Functions)** | لا تُستخدم هنا في الجذر — لا قلق إضافي |

---

## 2) تقليل بناءات Netlify (الأهم)

### أ) تعطيل ما لا تحتاجه من لوحة Netlify

**Site settings → Build & deploy → Continuous Deployment → Deploy contexts**

- **Deploy Previews:** إن لم تكن تستخدم معاينة كل Pull Request، عطّلها أو اجعلها **Manual** إن وُجدت.  
  كل PR كان يولّد بناءً إضافياً = استهلاك دقائق بدون فائدة لك.
- **Branch deploys:** إن لم تنشر فروعاً جانبية، عطّل النشر التلقائي للفروع غير `main`.

(الأسماء قد تختلف قليلاً بين إصدارات الواجهة؛ ابحث عن *Deploy previews* و *Branch deploys*.)

### ب) تخطي البناء عند تعديلات «توثيق فقط»

في رسالة الـ **commit** على الفرع الذي يبني منه Netlify، أضف:

```text
[skip netlify]
```

مثال:

```text
docs: تحديث README فقط [skip netlify]
```

المشروع مضبوط في **`netlify.toml`** بحيث **لا يُشغَّل بناء** Netlify عند وجود هذا النص في آخر رسالة commit.

> استخدمه عندما تغيّر ملفات **Markdown / توثيق** فقط ولا تحتاج إعادة نشر الملفات الثابتة.

### ج) لا ترفع ملفات ضخمة بلا حاجة

الصور والفيديو والأرشيفات تزيد **الرفع + النطاق الترددي**. اضغط الصور واحذف ما لا يُستخدم.

---

## 3) GitHub Actions — تقليل الدقائق

- **المستودع العام (Public):** غالباً لا يُشترط القلق من حدود Actions للمشاريع الصغيرة.
- **المستودع الخاص:** الحصة محدودة — راقب **Settings → Billing → Actions**.

في هذا المشروع:

- سير **CI** لا يعمل عندما يتغيّر **Markdown وملفات توثيق فقط** (انظر `paths-ignore` في `.github/workflows/ci.yml`).
- سير **Deploy Netlify** يدوي فقط (`workflow_dispatch`) — لا يستهلك شيئاً حتى تشغّله أنت.

تمت إضافة **إلغاء التشغيل المتزامن** (`concurrency`) حتى لا تُكدّس طلبات بناء قديمة.

## 4) ملخص سريع

| تريد… | افعل… |
|--------|--------|
| تعديل README فقط | `commit` + `[skip netlify]` + ادفع؛ **CI** لن يعمل لملفات `.md` فقط |
| تعديل كود أو بيانات الجهات | ادفع عادياً — بناء واحد منطقي |
| معاينة PR | فعّل Deploy Previews فقط إذا احتجتها |
